# RNN_PETR4_keras
Time series for PETR4 stock prices prediction.

## Getting Started

RNN and LSTM model to predict PETR4 stock price.

### Prerequisites

```
    pandas==0.24.2
    matplotlib==3.0.3
    scikit-learn==0.20.3
    numpy==1.16.4
    Keras==2.2.4
```

### Installing

Download the csv file;
Choose the right .ipynb file and run it.

For instance:
```
   recorrente_multiplas_saidas.ipynb 
```
